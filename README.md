# Dark attitudes
A modified version of attitude, a game made by Rofwell

Play it now at 

Play the original at [rofwell.github.io/attitude](https://rofwell.github.io/attitude)

## about the game
The aim of this game is to complete all the levels. To do so, pass a level. If you fail twice, you'll have to start again!

## how to play
Complete a level by collecting every coin. Make sure to avoid the hazards along the way.

## controls
### in-game
Press **space** on the start screen to begin playing.

| key(s)          | action        |  
| --------------- | ------------- |
| x               | die           |
